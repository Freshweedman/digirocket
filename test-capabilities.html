Teste de Duplicação

Este arquivo vai contar quantas vezes "Serviços que fornecemos" aparece na página principal.

Abra o site principal e cole este script no console do navegador:

const bodyText = document.body.innerText;
const searchText = "Serviços que fornecemos";
const count = (bodyText.match(new RegExp(searchText, "g")) || []).length;
console.log(`"${searchText}" aparece ${count} vez(es)`);
alert(`"${searchText}" aparece ${count} vez(es)`);
